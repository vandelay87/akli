A personal website made by Akli Aissat about web development and programming.

## 🚀 Quick start

1.  **🏃 Run**

    Use the Gatsby CLI to run locally.

    ```shell
    # run Gatsby locally
    gatsby develop
    ```

1.  **🏗️ Build**

    Build for production.

    ```shell
    gatsby build
    ```
